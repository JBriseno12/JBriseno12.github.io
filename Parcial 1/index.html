Comandos
-ls

Descripcion:

Lista los archivos y directorios en el directorio actual.

Comando:

~$ ls

-l

Descripcion:

Muestra la lista en formato largo, incluyendo permisos, número de enlaces, propietario, grupo, tamaño, y fecha de última modificación para cada archivo.

Comando:

~$ l

-a

Descripcion:

Muestra todos los archivos, incluyendo aquellos ocultos

Comando:

~$ a

Crear Documento

Descripcion:

Crea un archivo vacío si el archivo no existe.

Comando:

~$ touch nombre_del_archivo

Crear Directorio

Descripcion:

Crea un directorio vacío si el directorio no existe.

Comando:

~$ mkdir nombre_del_archivo

Eliminar Documentos

Descripcion:

Elimina archivos.

Comando:

~$ rm nombre_del_archivo

Descripcion:

Elimina directorio vacio.

Comando:

~$ rmdir nombre_del_directorio

Nano:

Descripcion:

Editor de texto simple y sencillo de usar

Comando:

~$ nano nombre_del_archivo

Guardar:

Ctrl + O


Salir:

Ctrl + X

Mint

Descripcion:

Linux Mint es un sistema operativo muy moderno; su desarrollo inició en 2006. Sin embargo, está basado en capas de software muy maduras y aprobadas, incluyendo el kernel (núcleo) Linux, las herramientas GNU y el escritorio Cinnamon.
