Apuntes de Linux
Nano:

~$ nano nombre_del_archivo

Salir y Guardar:

Ctrl + x

Descripcion:

Editor de texto simple y sencillo de usar

Micro:

~$ micro nombre_del_archivo

Guardar:

Ctrl + s

Salir:

Ctrl + q

Descripcion:

Editor de texto mas inteligente, en el cual identifica si estas haciendo un codigo.

Vim:

~$ vim nombre_del_archivo

Descripcion:

Es un editor de texto minimalista de linux.

Cat:

~$ cat nombre_del_archivo

Descripcion:

Permite previsualizar el contenido del archivo, sin abrirlo con un editor de texto
